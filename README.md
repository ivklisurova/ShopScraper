## Shop Scraper

### Description

Shop Scraper is a minimal script for demonstration purposes where information from provided links is extracted and parsed in a requested format.

### Requirements

- Python 3.8+

### Installation
#### Clone the repository

1. Copy the link from Github Repository.
2. Change the current working directory to the location where you want the cloned directory.
3. Open GitBash and clone repository

```bash
$ git clone https://github.com/ivklisurova/ShopScraper.git
```
4. Install requirements
```bash
pip install -r requirements.txt
```
### Usage

To locate the spider navigate to shopscraper directory

```bash
cd .\shopscraper\ 
```

Run the spider with following command

```bash
scrapy crawl  <spider name>
```
